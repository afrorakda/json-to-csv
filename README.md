# #32 JSON to CSV Converter

A hyper-minimalist tool to transform structured JSON arrays into clean, Excel-ready CSV format.

## Core Philosophy
* **1 Tool / 1 Action**: Converts JSON data to CSV instantly without any extra steps.
* **Smart Processing**: Automatically handles nested commas and double quotes with robust escaping.
* **Privacy Focused**: Browser-only execution. Your data never leaves your device.

## How to Use
1. Paste your **JSON Array** into the input field.
2. Click **Convert to CSV**.
3. Use **Copy** to grab the text or **Download** to save it as a `.csv` file.
4. Click **Clear (×)** to start over.

## Features
* **Subtitle Guide**: "Enter JSON array to convert into CSV format."
* **Dynamic UI**: Result area remains hidden until conversion is successful.
* **Mobile Ready**: Optimized for touch interaction with 15px rounded corners.

## Privacy & Performance
* No login / No tracking / No ads / Browser-only.
* Lightweight code for fast loading on any device.
